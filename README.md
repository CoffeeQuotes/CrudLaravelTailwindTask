# Crud Course Task

Build with Laravel 8 and Tailwind.

## Installation

1. Clone the repository into your local system folder.
2. Create a Database.
3. Change the .env file according to your database configuration.
4. Run the below commands.

```bash
npm install
npm run dev
php artisan migrate
php artisan serve
```
